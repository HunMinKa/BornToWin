### LG전자 PMS 구축

  프로젝트 목표
  - 제품 개발의 필요한다수의 ALM의 통합 및 효율성 추구을 위한 통합 관리 시스템 구축을 하여 사용자의 편의성 향상을 통한 업무생산성 개선을 위한 통합 사이트 개발
  
  개발기간 : 2018.10 - 2019.01
  
  개발 목록
  1. CodeBeamer와 Jira 서로다른 ALM의 개발 관리 문서 통합을 위한 DataLink 기능 개발
  - JIRA, CodeBeamer 연동을 위한 REST-API 기능
  - 코드관리를 통한 다른 ALM 사용시 Datalink Settings 기능
  - ALM Task간의 연동을 위한 Mapping 기능
  - 연동된 정보의 데이터 시각화를 위한 Pivot Data Graph 현황
  
  사용기술
  - FrontEnd: Javascript
  - BackEnd: .NET framework 3.5
  - Server: WindowsServer
  - SQL: MS-SQL

### 혼다코리아 SharePoint 온라인 시스템 구축

  4. 부서 및 팀별 분석을 통한 최적화된 업무 기능및 전자결재 기능 개발

### 실리콘 마이터스 PMS 구축

  프로젝트 목표
  - 제품 개발 프로젝트의 작업 절차 개선과 프로젝트의 진행 및 목표에 대한 효율적인 관리를 위한 관리 시스템 구축
    
  개발기간 : 2020.02 - 2020.06
  
  개발 목록
  1. 제품 개발 리스크 관리를 위한 개발 프로세스 관리 (NPDP)
  - 데이터 연동을 위한 REST-API
  - 다양한 프로젝트의 기능 연결을 위한 WBS 템플릿 및 자동설정
  - 제품별 개발 DIAGRAM 설정
  - 프로젝트 작업,이슈 관리
  - 회의소집 메일링 ,회의록
  - 프로젝트 별 제품 Rev 관리
  2. 제품 개발에 필요한 업무 결재 및 히스토리를 위한 전자 결재
  - 프로젝트 등록 요청서
  - 프로젝트 Hold/Hold Release/Drop 상태 변경 요청서
  - 프로젝트 변경 요청서 (일정변경, PM/PL 변경, 구성원 변경)
  - 작업 산출물 등록 및 버전 관리
  3. 제품 개발의 진행상황 및 데이터 시각화를 위한 대시보드
  - 프로젝트 별 진행 현황
  - BU 별 프로젝트 진행 통계
  - NPDP 기능을 통한 리스크 현황
  4. 기타
  - MS SharePoint Server 2019 및 Project Server 2019 환경 구축
  - 실리콘마이터스 사내 Active Directory 및 그룹웨어 시스템의 사용자 정보 연동
  - 공지사항 관리 기능 및 사용자 노출 기능
  - 이전 시스템(DPMS 1.0) Data Migration

  사용기술
  - FrontEnd: Javascript,Devexpress,d3.js,canvas.js
  - BackEnd: .NET framework 3.5
  - Server: WindowsServer
  - SQL: MS-SQL

### 실리콘 마이터스 TCS 구축

  프로젝트 목표
  - 이전 프로젝트에서 개발했던 PMS와의 연동을 통해 효율적인 인적 관리(M/M)를 위한 Tmesheet 관리 시스템 구축
  
  개발기간 : 2020.07 - 2020.09
  
  개발목록
  1. PMS와 연동을 통하여 프로젝트별 인원 관리 기능
  - 데이터 연동을 위한 REST-API
  - PMS와 TCS 연동 통한 프로젝트의 인원 관리
  - PMS와 TCS 간의 인원별 다대다 연동을 위한 코드 관리
  2. Timesheet 관리
  - 인원별 프로젝트별 M/M 입력 기능
  - 인원별 프로젝트별 M/M 승인 기능
  - 외부 개발 인원 등록을 위한 인사 관리
  - 엑셀양식을 통한 데이터 업로드/다운로드 기능
  3. 대시보드
  - BU 별 프로젝트 M/M 통계 현황
  - 사용자별 프로젝트 M/M 통계 현황
  ● 2020.7 - 2020.9
  - 제품별 프로젝트 M/M 통계 현황
  - 대시보드에 출력된 M/M 데이터 엑셀 다운로드 기능
  4. 기타
  - 사용자,그룹,메뉴,코드 관리 (사이트 관리) 기능
  - 그룹웨어 사용자 정보 연동
  - 기존 엑셀 문서로 관리해왔던 Timesheet Data Migration

  사용기술
  - FrontEnd: Javascript,Devexpress
  - BackEnd: .NET framework 3.5
  - Server: WindowsServer
  - SQL: MS-SQL
